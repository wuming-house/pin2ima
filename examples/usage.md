# Pin2IMA 使用示例

## 保存文章

**输入：**
```
https://mp.weixin.qq.com/s/xxx 保存到知识库
```

**处理过程：**
1. 识别为文章链接（公众号）
2. 查找目标知识库（默认：「公众号文章知识库」）
3. 调用 `import_urls` 保存链接
4. 返回保存结果

**输出：**
```
📥 正在保存到「公众号文章知识库」…

✅ 已添加到知识库「公众号文章知识库」✓
```

---

## 保存视频

**输入：**
```
https://www.bilibili.com/video/BV1xx411c7x 收藏到知识库
```

**处理过程：**
1. 识别为视频链接（B站）
2. 提取视频 ID：BV1xx411c7x
3. 通过 B站 API 拉取字幕
4. AI 生成 2-3 句摘要
5. 创建 IMA 笔记保存摘要
6. 将笔记挂载到知识库
7. 同时保存视频 URL 入口

**输出：**
```
📥 正在保存到「公众号文章知识库」…

🎬 视频快照
标题：为什么 AI 视频生成在 2026 年爆发？
来源：B站 · up主: xxx
字幕摘要：从算力成本下降、开源模型成熟、商业场景验证三个角度
          分析了 AI 视频生成爆发的底层逻辑。

✅ 笔记已保存 → 在知识库中搜索「为什么 AI 视频生成在 2026 年爆发 - 选题笔记」即可查看
```

## 跨知识库搜索

**输入：**
```

```

## 今日选题速览（定时任务）

**在 WorkBuddy 中创建定时任务（automation）的 prompt：**
```
从「公众号文章知识库」中列出最近 7 天收藏的文章，
按话题分组，输出一份选题速览报告。
重点标注哪些话题出现频率高、哪些角度值得跟进。
```

**输出示例：**
```
📊 本周选题速览

🔥 热门话题
1. AI 视频生成（3 篇）← 高频
   - 文章A ... 
   - 文章B ...

2. 自媒体运营（2 篇）
   ...
```
